# South Maroubra Dolphins Winter Swimming Club

## Installation

```shell
git clone git@github.com:robertlove/south-maroubra-dolphins.git
cd south-maroubra-dolphins
```

## Usage

```shell
bundle
jekyll build
jekyll serve
```
